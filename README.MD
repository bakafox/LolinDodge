## Connection Pinout:

**For display:**
- VCC <—> 3V (food 3.3 V)
- GND <—> G  (ground)
- CS  <—> D8 (GPIO15, HSPI_CS)
- RST <—> D3 (GPIO0)
- A0  <—> D4 (GPIO2)
- SCK <—> D5 (GPIO14, HSPI_SCK)
- SDA <—> D7 (GPIO13, HSPI_MOSI)
- LED <—> 3V (food 3.3 В)

**For gamepad:**
- 30J: A0 (ADC)
- 0J: 3V (food 3.3 В)
- -1J: G (ground)
